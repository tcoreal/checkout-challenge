# Purge Order

Web API Call for removing all items from order.

## Specification:

 * **Url Path:** /api/orders/purge/{orderId} 

 * **Method:** POST 
 
 * **Input Parameters:**
   * url part -  orderId: string
   
 * **Output:** None
